Brew Journal (Coffee calendar PWA) — No Mac required
====================================================

What this is
------------
A CoffeeMood-like (vibe-wise) personal coffee calendar you can run on iPhone/iPad without Xcode.
- Calendar view (tap a day to pin a cup)
- Coffee picker (3x4 grid)
- Entry editor (note, place, favorite, optional photo)
- Bookmarks screen
- Settings: export/import JSON backup, export month poster (SVG), clear data
- Offline after first load (service worker cache)

How to use (fastest)
--------------------
1) Host it for free on GitHub Pages:
   - Create a GitHub account (if you don't have one).
   - Create a new repository, e.g. 'brewjournal'.
   - Upload the files in this folder to the repo root.
   - Repo Settings → Pages → Deploy from branch → main / (root).
   - Wait for the Pages URL (https://<user>.github.io/<repo>/)

2) On iPhone/iPad:
   - Open the Pages URL in Safari.
   - Share button → "Add to Home Screen"
   - Launch it from the Home Screen like an app.

Backup
------
Settings → Export JSON backup.
If you ever worry about losing history, export occasionally.

Icons
-----
Icons are original SVG drawings (CoffeeMood-like vibe, not copied).
You can further customize the palette in app.css.
